# ColorPicker


이게 무엇인가요?    
-----------
모니터에 출력되고 있는 특정한 부분의 색상값을 추출해주는 응용 프로그램입니다.  
색상값을 얻기 위해 화면 갈무리(Print Screen)를 할 필요도 없고, 어떤 그래픽 프로그램을 켤 필요도 없습니다.  
  

어떻게 사용하나요?    
-----------
그저 프로그램을 켜고 `(grave accent, 억음 부호) 키만 누르면 됩니다.  
  

프로그램 버전 현황    
-----------

사실상 최종 버전.
